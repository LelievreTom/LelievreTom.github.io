Mes Projets

Jeu ludo-pédagogique

Le but du projet était de créer un jeu ludo-pédagogique en java et en groupe de 2. Le Jeu était un escape game dans un manoir abandonné, où l'utilisateur se baladé salle en salle en répondant à des questions. Les graphismes étaient en code ASCII dans le terminal directement.

Exploitation d'une base de donnée ParcourSup

Dans ce projet en équipe de 2, nous devions organiser des données publiques de ParcourSup dans une base de donnée pour ensuite faire des requêtes pour faire des calculs. Par exemple : calculer le nombre d'étudiants ayant candidaté dans une certaine formation.
